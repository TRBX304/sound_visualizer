# 🎵 Sound Visualizer（Chrome拡張）

Web Audio API を使用したフルスクリーン型オーディオビジュアライザーです。  
タブの音声をリアルタイムで解析し、周波数スペクトラム・パーティクル・波エフェクトを描画します。

---

## ✨ 主な機能

- 🎧 タブ音声のリアルタイムキャプチャ
- 📊 FFT による周波数解析（AnalyserNode）
- 🔥 二乗平均（RMS的処理）によるピーク強調
- 🌈 HSLグラデーション＋グロー付きスペクトラムバー
- 🌊 全体音量に反応する波エフェクト
- ✨ 背景パーティクル演出
- 🖥 フルスクリーンキャンバス表示
- ⚡ requestAnimationFrame による滑らかな描画
